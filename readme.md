# IONIC ANGULAR APP

This project generate by @ionic/angular version 8

## Run Locally

Clone the project

```bash
  git clone https://github.com/kraiwith/ionic-angular-app.git
```

Go to the project directory

```bash
  cd ionic-angular-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
